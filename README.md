#  Finance Dashboard UI

A responsive Finance Dashboard UI built using React + Tailwind CSS to help users track financial activity, explore transactions, and understand spending patterns through charts and insights.


---

#  Features

* Summary cards (Balance, Income, Expenses)
* Time-based visualization (Line Chart)
* Category-based visualization (Pie Chart)
* Transactions table with search functionality
* Simulated role-based UI (Admin / Viewer)
* State management using Context API
* LocalStorage data persistence
* Fully responsive layout

---

#  Tech Stack

* React
* Tailwind CSS
* Recharts
* Context API
* Vite
